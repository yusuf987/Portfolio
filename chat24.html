Chat 24
Project : Chat 24 (Personal Project)
URL : https://chat24.in
My Role : UI Developer, Frontend Engineer
Technology : React, TypeScript, HTML5, CSS3, Socket.IO, Node.js

Project Description :

Chat 24 is a real-time communication platform focused on delivering a fast, intuitive, and engaging user experience. The application was designed with a UI-first approach, emphasising responsiveness, accessibility, and smooth real-time interactions.

I designed and built the complete frontend interface using React and TypeScript, transforming UI concepts into reusable components with consistent styling and interaction patterns. The UI supports real-time messaging, voice/video calling, and interactive features, while maintaining high performance and visual clarity.

Advanced CSS techniques were used to create fluid animations and transitions, ensuring a polished, modern look and feel across devices and screen sizes.
